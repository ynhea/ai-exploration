# 🤖 CompassBot
> AI Engineer Job Experience (Week 3)

FastAPI와 LLM을 활용하여 **사내 AI Assistant(CompassBot)** 를
MVP부터 실제 서비스 수준까지 발전시키는 프로젝트입니다.

이 프로젝트는 AI 이론을 공부하는 것이 아니라,
실제 AI 서비스를 설계하고 개선하며 배포하는 과정을 경험하는 것을 목표로 합니다.

---

# 📖 프로젝트 소개

보안 엔지니어(1주), 인프라 엔지니어(2주) 직무 체험에 이어

이번에는 **AI Engineer**의 업무를 직접 경험해보기 위해 진행한 프로젝트입니다.

가상의 스타트업 **CompassLab** 의 AI Engineer 인턴이 되어

사내 AI Assistant **CompassBot** 을

MVP → RAG → Agent → MCP → Deployment까지 단계적으로 발전시킵니다.

---

# 🎯 프로젝트 목표

- LLM 기반 AI 서비스 개발 경험
